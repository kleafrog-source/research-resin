# Research Resin - Ion Exchange Analysis

A Streamlit application for simulating and analyzing ion exchange in polymer resins.

## Features
- Property prediction for new ions
- Advanced analysis and visualization
- Simulation of ion exchange processes

## Setup
```bash
# Clone the repository
git clone [https://github.com/kleafrog-source/research-resin.git](https://github.com/kleafrog-source/research-resin.git)
cd research-resin

# Install dependencies
pip install -r requirements.txt

# Run the application
streamlit run app.py
